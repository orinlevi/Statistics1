# סטטיסטיקה לפסיכולוגיה א' - סיכום קורס

סיכום מקיף של קורס סטטיסטיקה לפסיכולוגיה א' באוניברסיטת תל אביב.

## צפייה באתר

האתר זמין בכתובת: https://orinlevi.github.io/Statistics_1_sum/

## תוכן

הסיכום כולל 13 יחידות:

1. מושגי יסוד וסולמות מדידה
2. מדדי מרכז
3. מדדי פיזור
4. מדדי מיקום יחסי ומדדי קשר
5. קומבינטוריקה
6. הסתברות
7. מעבר ממדגם לאוכלוסייה
8. ייצוגים גרפיים, התפלגות וצפיפות
9. ההתפלגות הנורמלית
10. התפלגות הדגימה
11. בדיקת השערות סטטיסטיות
12. טעויות סטטיסטיות ועוצמת המבחן
13. גודל אפקט ורווח סמך

בנוסף: סיכום כללי וטבלת Z.

## פיתוח מקומי

### דרישות
- Ruby 3.x
- Jekyll 4.x
- XeLaTeX (לקימפול PDF)

### התקנה

```bash
bundle install
```

### הרצה מקומית

```bash
bundle exec jekyll serve
```

האתר יהיה זמין ב-`http://localhost:4000/statistics_1_sum/`

## קימפול PDF

```bash
xelatex main.tex
```
