# טבלת Z

טבלת Z מציגה את השטח מתחת לעקומת ההתפלגות הנורמלית הסטנדרטית.

## כיצד לקרוא את הטבלה

הטבלה נותנת את \(P(Z < z)\) - השטח משמאל לערך Z.

- **שורה**: הספרה השלמה והעשירית של Z
- **עמודה**: המאית של Z

## כללי שימוש

### מציאת שטח מתחת לערך

\[P(Z < z) = \text{ישירות מהטבלה}\]

### מציאת שטח מעל ערך

\[P(Z > z) = 1 - P(Z < z)\]

### מציאת שטח בין שני ערכים

\[P(z_1 < Z < z_2) = P(Z < z_2) - P(Z < z_1)\]

### סימטריה

\[P(Z < -z) = 1 - P(Z < z)\]

\[P(Z > -z) = P(Z < z)\]

## דוגמאות חישוב

### דוגמה 1: P(Z < 1.5)

מהטבלה: \(P(Z < 1.5) = 0.9332\)

### דוגמה 2: P(Z > 1.5)

\[P(Z > 1.5) = 1 - P(Z < 1.5) = 1 - 0.9332 = 0.0668\]

### דוגמה 3: P(-1 < Z < 1)

\[P(-1 < Z < 1) = P(Z < 1) - P(Z < -1)\]
\[= 0.8413 - 0.1587 = 0.6826 \approx 68\%\]

### דוגמה 4: P(Z < -2)

באמצעות סימטריה:
\[P(Z < -2) = 1 - P(Z < 2) = 1 - 0.9772 = 0.0228\]
