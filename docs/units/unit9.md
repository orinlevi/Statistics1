# יחידה 9: ההתפלגות הנורמלית

יחידה זו עוסקת בהתפלגות הנורמלית, שהיא מודל תאורטי מרכזי בסטטיסטיקה. הדגש הוא על הבנת צורת ההתפלגות, משמעות הפרמטרים שלה, והקשר בין שטח תחת העקומה להסתברות.

## מהי התפלגות נורמלית

התפלגות נורמלית היא התפלגות רציפה, סימטרית ופעמונית, המתוארת על ידי שני פרמטרים:

- \(\mu\) – תוחלת (ממוצע)
- \(\sigma\) – סטיית תקן

### מאפיינים מרכזיים

- סימטריה סביב \(\mu\)
- ממוצע = חציון = שכיח
- זנבות אינסופיים (אך השכיחות בהם קטנה מאוד)

## משמעות הפרמטרים

### התוחלת \(\mu\)

קובעת את מיקום ההתפלגות על ציר ה־X.

### סטיית התקן \(\sigma\)

קובעת את פיזור ההתפלגות:

- \(\sigma\) גדולה → התפלגות רחבה ושטוחה
- \(\sigma\) קטנה → התפלגות צרה וגבוהה

## שטח כהסתברות

### פונקציית צפיפות אינה הסתברות

פונקציית הצפיפות \(f(x)\) אינה נותנת הסתברות לערך בודד. היא מתארת את **קצב הצטברות ההסתברות** סביב ערך מסוים.

לכן:

- \(f(x)\) יכולה להיות גדולה מ־1
- \(f(x)\) אינה הסתברות
- רק **שטח** מתחת לגרף מייצג הסתברות

בהתפלגות נורמלית:

<div class="formula-box" markdown>

\[\hebrew{השטח מתחת לעקומה} = \hebrew{הסתברות}\]

</div>

### דגשים

- סך כל השטח = 1
- הסתברות לערך בודד = 0
- מחשבים הסתברויות רק עבור תחומים

## כלל האצבע (68–95–99.7)

<div class="example-box" markdown>

- כ־68% מהתצפיות בתחום \(\mu \pm \sigma\)
- כ־95% בתחום \(\mu \pm 2\sigma\)
- כ־99.7% בתחום \(\mu \pm 3\sigma\)

</div>

!!! warning "דגש למבחן"
    כלל האצבע הוא **קירוב**, לא חישוב מדויק.

## תקנון

### הרעיון

המרה של משתנה נורמלי כללי למשתנה נורמלי סטנדרטי.

<div class="formula-box" markdown>

\[Z = \frac{X - \mu}{\sigma}\]

</div>

### שימור הסתברויות תחת תקנון

התקנון הוא טרנספורמציה לינארית שאינה משנה הסתברויות אלא רק את סולם המדידה.

לכן מתקיים:

\[P(a \le X \le b) = P\left(\frac{a-\mu}{\sigma} \le Z \le \frac{b-\mu}{\sigma}\right)\]

!!! note "דגש"
    התקנון אינו משנה את סדר הערכים ולכן אינו משנה אחוזונים.

## ההתפלגות הנורמלית הסטנדרטית

מסומנת \(Z \sim N(0,1)\).

### למה מתקננים

- כדי להשתמש בטבלת Z
- כדי להשוות בין משתנים שונים

## טבלת Z

טבלת Z נותנת את השטח מהממוצע (0) ועד לערך Z בעמודה B. כדי לקבל את השטח הכולל משמאל ל־Z חיובי מחשבים:

<div class="formula-box" markdown>

\[P(Z < z) = 0.5 + B\]

</div>

### קריאה בסיסית

- שורה – ספרות שלמות + עשירית
- עמודה – מאיות

## מקרים קלאסיים לחישוב

### מתחת לערך

\(P(Z < z)\) ⇒ ישירות מהטבלה

### מעל לערך

\[P(Z > z) = 1 - P(Z < z)\]

### בין שני ערכים

\[P(z_1 < Z < z_2) = P(Z < z_2) - P(Z < z_1)\]

### סימטריה

\[P(Z < -z) = 1 - P(Z < z)\]

## אחוזונים בהתפלגות נורמלית

### רעיון

אחוזון הוא ערך שמתחתיו נמצאים \(p\%\) מהנתונים.

### שימוש בטבלה

1. מוצאים בטבלה שטח קרוב ל־\(p\)
2. קוראים את ערך Z
3. מבצעים תקנון הפוך:

<div class="formula-box" markdown>

\[X = \mu + Z\sigma\]

</div>

## טעויות נפוצות במבחן

<div class="warning-box" markdown>

- לשכוח ש־Z-table נותנת שטח משמאל
- להתבלבל בין מעל ומתחת
- לא לבצע תקנון לפני שימוש בטבלה
- לחשוב שכל משתנה הוא נורמלי

</div>
