# סיכום הקורס

סיכום מקיף של כל הנושאים בקורס סטטיסטיקה לפסיכולוגיה א'.

## חלק א': סטטיסטיקה תיאורית

### יחידה 1: מושגי יסוד וסולמות מדידה

- **משתנה**: תכונה או תופעה שיכולה לקבל ערכים שונים
- **משתנה בדיד**: ערכים ניתנים למנייה
- **משתנה רציף**: אינסוף ערכים אפשריים
- **ארבעת הסולמות**: שמי → סדר → רווח → יחס

### יחידה 2: מדדי מרכז

- **שכיח**: הערך הנפוץ ביותר (מתאים לסולם שמי)
- **חציון**: הערך האמצעי (עמיד לקיצוניים)
- **ממוצע**: סכום חלקי מספר התצפיות (רגיש לקיצוניים)

### יחידה 3: מדדי פיזור

- **שונות**: \(S^2 = \frac{1}{n}\sum(x_i - \bar{X})^2\)
- **סטיית תקן**: \(S = \sqrt{S^2}\)
- **MAD**: ממוצע סטיות מוחלטות מהחציון
- **IQR**: טווח בין-רבעוני

### יחידה 4: מדדי מיקום יחסי וקשר

- **ציון תקן**: \(Z = \frac{x - \bar{X}}{S}\)
- **מתאם פירסון**: \(-1 \le r \le 1\)
- תקנון לא משנה את עוצמת הקשר

## חלק ב': הסתברות

### יחידה 5: קומבינטוריקה

- **פרמוטציה**: \(n!\)
- **וריאציה**: \(\frac{n!}{(n-k)!}\)
- **קומבינציה**: \(\binom{n}{k} = \frac{n!}{k!(n-k)!}\)

### יחידה 6: הסתברות

- **הסתברות מותנית**: \(P(A|B) = \frac{P(A \cap B)}{P(B)}\)
- **עצמאות**: \(P(A \cap B) = P(A) \cdot P(B)\)
- **נוסחת בייס**: \(P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}\)

## חלק ג': מעבר מתיאור להסקה

### יחידה 7: מדגם ואוכלוסייה

- **פרמטר**: גודל קבוע של האוכלוסייה (\(\mu, \sigma\))
- **סטטיסטי**: גודל מחושב מהמדגם (\(\bar{X}, S\))
- **תוחלת**: \(E(X) = \sum x \cdot P(X=x)\)

### יחידה 8: ייצוגים גרפיים וצפיפות

- בהיסטוגרמה: **שטח** = שכיחות (לא גובה)
- צפיפות יחסית: \(\frac{f_i}{n \cdot w_i}\)
- סכום כל השטחים = 1

### יחידה 9: ההתפלגות הנורמלית

- **תקנון**: \(Z = \frac{X - \mu}{\sigma}\)
- **כלל 68-95-99.7**
- שטח = הסתברות

### יחידה 10: התפלגות הדגימה

- **תוחלת**: \(\mu_{\bar{X}} = \mu\)
- **טעות תקן**: \(\sigma_{\bar{X}} = \frac{\sigma}{\sqrt{n}}\)
- משפט הגבול המרכזי: \(n \ge 30\)

## חלק ד': בדיקת השערות

### יחידה 11: בדיקת השערות

- \(H_0\): השערת האפס
- \(H_1\): השערת המחקר
- **מבחן Z**: \(Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}}\)
- \(p \le \alpha\) → דוחים \(H_0\)

### יחידה 12: טעויות ועוצמה

- **טעות סוג I** (\(\alpha\)): דוחים \(H_0\) שנכונה
- **טעות סוג II** (\(\beta\)): לא דוחים \(H_0\) שגויה
- **עוצמה**: \(1 - \beta\)

### יחידה 13: גודל אפקט ורווח סמך

- **Cohen's d**: \(d = \frac{\bar{X}_1 - \bar{X}_2}{s_{pooled}}\)
- **רווח סמך**: \(\bar{X} \pm Z_{\alpha/2} \cdot \frac{\sigma}{\sqrt{n}}\)

## נוסחאות מרכזיות

| נוסחה | שימוש |
|-------|-------|
| \(\bar{X} = \frac{\sum x_i}{n}\) | ממוצע |
| \(S^2 = \frac{\sum(x_i - \bar{X})^2}{n}\) | שונות |
| \(Z = \frac{X - \mu}{\sigma}\) | תקנון |
| \(\sigma_{\bar{X}} = \frac{\sigma}{\sqrt{n}}\) | טעות תקן |
| \(P(A|B) = \frac{P(A \cap B)}{P(B)}\) | הסתברות מותנית |
